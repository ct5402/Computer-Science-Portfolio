What is HTML?

HTML stands for HyperText Markup Language. It is usally used to create websites. Your are able to place images, link pages on the website, link other websites, and finally place simple headings and titles. HTML was invented in 1990 by Tim Berners-Lee. He was working at CERN a place where scientist meet and discuss things, he was trying to think of something where scientist and researchers were able to collabarate together. He then came up with the idea of the World Wide Web! HTML was created for a way to allow people to create their own websites. Although without CSS HTML is not very pleasing to the eye!

CSS

CSS stands for Cascading Style Sheets. CSS is what is used to style HTML and other markup languages.CSS and HTML both use tags but CSS tags are mainly used to style things and organize them.CSS was not released the same time HTML was because at first HTML was created to share your ideas not really show your creativity.People then started to complain that they didn't have enough influence over their webpage and how it look so people thought of the idea of CSS, a way to style their website to their liking. I think its something very important to use while using HTML because it allows you to give your website personality. CSS allows to better organize HTML and making it look cooler.
